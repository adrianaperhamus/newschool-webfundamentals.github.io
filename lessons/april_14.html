What's a function?

Functions are useful for executing code that happens repeatedly, step-by-step.

It's annoying to re-type these variables every time the income changes. We can use a function to produce a different output (max monthly rent) given a different input (income).

Here's the basic syntax for a function

We can name our functions like we would variables and pass it an input (income).

Now that we have our function with our input, let's do our calculations and use the keyword "return" to return the result (output)

After declaring our function, we can use it to calculate different max rents based on income. It will return a different result every time the function runs.

Let's writer another function together, to produce the max rent message.

Now, we will use our maxRent variable to construct a string about how much rent to spend

Every time the input changes, the output changes

You can also store the result of functions in variables

We can then pass that variable as an input to our message function

Quick note about functions - they can take as many inputs as you want. You just have to name them differently

Sometimes we want to conditionally return an ouput based on the input. Like the bouncer or grade calculator we created.

JavaScript has conditionals for executing this type of code.

If the condition is true, it returns the result in the if block.

If the condition is false, it returns the result in the else block

You can use conditionals to return different results in a function.

You can add more than two conditionals, but using the "else if" syntax
