So, functions are great for performing the same task many times over and conditionals help make decisions, but what about making a webpage interactive?

The purpose of jQuery is to make it much easier to use JavaScript on your website.

  • HTML & DOM manipulation
  • CSS manipulation
  • Listen to events
  • Effects and animations

In the same way that Bootstrap gave you CSS to style a page faster, jQuery gives you functions to interact with a webpage without having to write difficult "vanilla" JavaScript

When a web page is loaded, the browser creates a Document Object Model of the page.

The HTML DOM model is constructed as a tree of Objects that you can interact with.

Imagine it as a tree-like representation of your HTML

The DOM acts as a family-tree of HTML nodes
